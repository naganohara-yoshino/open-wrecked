## About the Project

A yuri-themed visual novel about an all-girl band, featuring a bad storyline between the female characters.
The story ends in tragedy.

This project is developed using the [Ren'Py](https://www.renpy.org/) engine.

---


少女バンドを題材にした百合系ビジュアルノベルですが、登場する少女たちの物語はあまり出来の良いものではありません。
物語は悲劇的な結末で終わります。

本プロジェクトは [Ren'Py](https://www.renpy.org/) エンジンを使用して開発されています。

---

這是一款以少女樂團為題材、具有百合元素的視覺小說，但角色之間的劇情相當糟糕。
故事最終以悲劇收場。

本專案基於 [Ren'Py](https://www.renpy.org/) 引擎開發。
